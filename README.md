# navegador
Estudo de um filtros estáticos e dinâmicos utilizando AngularJS

## Pré-requisitos
* NodeJS

## Setup
* npm install
* bower install
* ./node_modules/protractor/bin/webdriver-manager update

## Executar
* npm test
ou
* npm run e2e
